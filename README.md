# Beat Generator 🎵

A browser-based drum machine and step
sequencer using the Web Audio API.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- 16-step sequencer with 6 drum tracks
- 5 genre presets: Hip-Hop, Rock, Electronic,
  Reggae, Jazz
- BPM control (60–200)
- Master volume + reverb + swing controls
- Per-track volume sliders
- Random beat generator
- Real-time audio visualizer
- Works entirely in the browser (no backend)

## Instruments
Kick, Snare, Hi-Hat, Open HH, Clap, Perc
All synthesized via Web Audio API oscillators
and noise generators

## Tools Used
- Python, Streamlit, HTML5 Web Audio API,
  JavaScript

## How to Run Locally
pip install streamlit
streamlit run app.py
